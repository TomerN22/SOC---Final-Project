# SOC Final Project

## Project Overview
This repository contains my final SOC (Security Operations Center) project.  
The project focuses on security monitoring, log analysis and incident detection.

<div dir="rtl">

## תיאור הפרויקט
זהו פרויקט גמר במסגרת מקצה SOC.
בפרויקט בוצע ניתוח לוגים וזיהוי פעילות חשודה במטרה לדמות עבודת אנליסט SOC.

## מטרות הפרויקט
- תרגול עבודת אנליסט SOC
- ניתוח לוגים ממערכות שונות
- זיהוי פעילות חשודה או זדונית
- חקירת אירוע אבטחה והסקת מסקנות

## כלים וטכנולוגיות
בפרויקט נעשה שימוש בכלים הבאים:

- Splunk (SIEM)
- Sysmon
- Windows Event Logs

## תהליך החקירה
1. איסוף וניתוח לוגים ממקורות שונים
2. זיהוי פעילות חריגה במערכת
3. שימוש ב-SIEM לצורך חיפוש וחקירת אירועים
4. ניתוח הממצאים והסקת מסקנות

## תכולת הפרויקט
במאגר זה ניתן למצוא את קובץ הפרויקט המלא הכולל את תהליך החקירה והממצאים.
